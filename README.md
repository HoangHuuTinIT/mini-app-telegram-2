# Đây là 1 repo chứa mini-app telegram đã được tích hợp 1 số SDK hỗ trợ sẵn

## Clone Repository

```bash
git clone https://github.com/HoangHuuTinIT/mini-app-telegram-2.git
```

---

## Build mini-app
### Sau khi clone repo về, thực hiện các bước sau
1. Mở Terminal
2. Di chuyển đến vị trí đặt repo
3. Chạy lệnh npm install
4. Sau khi chạy xong lệnh npm install chạy tiếp lệnh npm run build

- Sau khi chạy xong lệnh npm run build,thư mục dist sẽ được tạo ra, thư mục này sẽ chính là thứ mà chúng ta cần để tích hợp mini-app vào 1 app android khác
